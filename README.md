# SupportAgent-OpenEnv 🤖

This project is a custom environment for AI Support Agents, built with **FastAPI** and deployed via **Docker** on Hugging Face.

### 🚀 Live Demo
[Click here to view the live environment on Hugging Face](https://huggingface.co/spaces/khushirai/Support-Agent-Openenv)

### 🛠️ Tech Stack
- **Backend:** FastAPI, Python 3.10
- **DevOps:** Docker, Hugging Face Spaces
- **Data Handling:** Pydantic, PyYAML

### 📌 Features
- Automated ticket prioritization.
- Real-time interaction through `/docs` (Swagger UI).
- Containerized for easy scaling.
